About This Project

×

Purpose: This project is a Docker Container Manager to visually manage containers, see CPU/RAM usage, and start/stop/delete them easily.

What I Learned: HTML, CSS, JS DOM manipulation, Fetch API, modern button designs, progress bars, and working with REST APIs.

Features Implemented:

  • Create new containers
  • Start/Stop/Delete containers
  • Live CPU/RAM usage visualization
  • Responsive design and modern buttons
  • About page for project documentation

How to Run: Run a backend API on http://localhost:8000 providing container info, then open this HTML file in your browser.
